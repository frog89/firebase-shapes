# **App Name**: Youtube Audiobook Manager

## Core Features:

- Output Folder Selection: Allow the user to select an output folder where generated audiobooks are stored.
- MP4 File Selection: Allow the user to select an MP4 file to convert into an audiobook.
- Audiobook Generation: Convert the selected MP4 file into an audiobook format and store it in the selected output folder.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke a calm, focused atmosphere suitable for long listening sessions.
- Background color: Light gray (#F0F4F8) for a clean and unobtrusive interface.
- Accent color: Deep orange (#FF7043) for the 'Execute' button and highlights, drawing attention to the primary action.
- Font: 'Inter', a sans-serif font that ensures readability and a modern feel for the interface.
- Responsive layout where text fields and the 'Execute' button resize horizontally with the main form.
- Simple and clear icons for folder and file selection to improve usability.
- Subtle progress animations during the audiobook generation process to keep the user informed.
